## 🛠️ How to use this repository

1. **Pick a module**: Start from `00-Cloud-Foundations` for foundational knowledge or jump to any topic.
2. **Read the README**: Each sub-folder will have a dedicated README with summary notes and interview-style questions.
3. **Practice**: Every module has hands-on concepts and practice scenarios.
4. **Interview Prep**: Review the "Interview Questions" section in each module to test your knowledge!

## ✨ Learning Principles

- **Don't just read, Type**: Always run the commands in your terminal.
- **Explain it like I'm five**: Try explaining a complex topic (like Kubernetes Ingress) in simple terms.
- **Build & Break**: The best way to learn is by troubleshooting errors.
